# JadiaRushilBookstore
Project involving various advanced topics in Java, creating a customer-to-store employee system.
Designed and developed a Bookstore Operations system to manage a Bookstore effortlessly, utilizing Polymorphism, Interfaces, ArrayLists, Overloading Constructors, Overriding Methods, Try-Catch, and Testing. 
This system was designed to support multiple user personas
As a customer, users can inquire about, and place an order for the books and other media
As a store manager, users can manage the store, they can add/update/delete items, view and manage stock in the store, place purchase orders to restock, etc.
